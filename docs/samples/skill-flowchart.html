Skill Explorer component — Stage 4c

Upgrade Path & Adjacent Skills

The upgrade-path flowchart lives inside every skill's detail panel in the Skills Explorer on the main site. It renders a sub-graph of prerequisite and derivative skills, tracing the ancestry upward and the learning path forward. This sample uses anthropic/skill-creator (tool-creation) as the subject.

Rendered by skill-explorer.js › renderFlowchart() + drawFlowEdges()

Click any node to lock its prerequisite chain. Node labels show the full skill name with contributor in red. Ghost nodes (dimmed) have no named implementation yet. Edge colors follow the source tier: cyan for Basic, purple for Extra, gold for Ultimate.
