## Overview

GitHub Project Management automates GitHub Projects board operations including issue creation from templates, milestone tracking, sprint board management, and label-based workflow automation. It integrates with GitHub Actions for automated issue triage and status updates.

## Key Capabilities

- **GitHub Projects board automation**: automated card creation, movement, and status tracking
- **Milestone tracking**: progress monitoring and deadline management across milestones
- **Issue templating**: structured issue creation from reusable templates
- **Label-based triage**: automated issue classification and routing via label workflows
